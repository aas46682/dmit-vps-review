## 性能实测：硬件这块 DMIT 确实没有水分

**CPU 性能**

DMIT 美国节点用的是 AMD EPYC 9000 系列，香港和东京用的是 EPYC 7003 系列。这两个系列都是服务器领域的高端配置。
